# Qwen Code

**Vendor:** Alibaba / Qwen Team
**Interface:** Terminal CLI
**Open Source:** Yes
**Repository:** https://github.com/QwenLM/qwen-code

## Overview

Qwen Code is Alibaba's CLI coding agent powered by the Qwen model family. It focuses on multilingual support (especially strong in Chinese) and offers a competitive alternative in the coding agent space.

## Models

| Model | Context | Best For |
|-------|---------|----------|
| Qwen 3 | 128K | General coding tasks |

## Key Strengths

- **Strong multilingual support** — excellent for Chinese + English codebases
- **Open weights** — model weights are publicly available
- **Growing ecosystem** — active community development
- **Competitive benchmarks** — strong SWE-bench scores

## Pricing

Free and open source. Can run locally or via API.

## Best For

- Developers working with Chinese-language codebases
- Teams wanting fully open-weight models
- Self-hosted deployments
