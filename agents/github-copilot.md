# GitHub Copilot

**Vendor:** GitHub / Microsoft
**Interface:** IDE extension + CLI
**Open Source:** No
**Website:** https://github.com/features/copilot

## Overview

GitHub Copilot is the most widely adopted AI coding assistant, available as an extension for VS Code, JetBrains, Neovim, and as a CLI tool. It recently added an "agent mode" for autonomous multi-file editing.

## Models

| Model | Context | Best For |
|-------|---------|----------|
| GPT-4o | 128K | General coding, completions |
| Claude 3.5 Sonnet | 200K | Complex reasoning tasks |
| Copilot's fine-tuned models | Varies | Inline completions |

## Key Strengths

- **Deepest IDE integration** — works in VS Code, JetBrains, Neovim, Xcode
- **Agent mode** — autonomous multi-file changes with terminal access
- **GitHub ecosystem** — native integration with Issues, PRs, Actions
- **Copilot Chat** — conversational coding within the IDE
- **Largest user base** — most third-party integrations and community resources
