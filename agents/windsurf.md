# Windsurf

**Vendor:** Codeium
**Interface:** IDE (VS Code fork)
**Open Source:** No
**Website:** https://codeium.com/windsurf

## Overview

Windsurf is Codeium's AI-native IDE, built as a VS Code fork with their proprietary Cascade model for multi-step agentic coding. It combines inline completions with an autonomous agent mode.

## Models

| Model | Context | Best For |
|-------|---------|----------|
| Cascade (proprietary) | Varies | Multi-step workflows |
| GPT-4o (optional) | 128K | General coding |
| Claude Sonnet (optional) | 200K | Complex reasoning |

## Key Strengths

- **Cascade flow** — proprietary multi-step reasoning that chains tool calls automatically
- **Inline completions** — fast tab-to-accept suggestions
- **Generous free tier** — substantial free usage included
- **Familiar IDE** — VS Code extension compatibility
