# Aider

**Vendor:** Paul Gauthier (independent)
**Interface:** Terminal CLI
**Open Source:** Yes
**Repository:** https://github.com/Aider-AI/aider

## Overview

Aider is an open-source AI pair programming tool that works with any LLM. It has deep git integration, automatically creates well-formatted commits, and supports a repo-map feature for codebase understanding.

## Models

Works with any LLM provider: OpenAI, Anthropic, Google, local models via Ollama/LM Studio, etc.

## Key Strengths

- **Model agnostic** — bring any LLM provider
- **Auto-commits** — creates clean git commits for each change
- **Repo map** — generates a map of the codebase for context
- **Linter integration** — runs linters after edits and auto-fixes
- **Voice mode** — speak your instructions
- **Leaderboard** — maintains LLM coding benchmarks at aider.chat

## Pricing

Free and open source. You pay only for LLM API usage.

## Setup

```bash
pip install aider-chat
aider
```
