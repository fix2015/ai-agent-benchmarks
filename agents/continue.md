# Continue

**Vendor:** Continue.dev
**Interface:** IDE extension (VS Code, JetBrains)
**Open Source:** Yes
**Repository:** https://github.com/continuedev/continue

## Overview

Continue is a fully open-source AI coding assistant that works as an IDE extension. Its key differentiator is model flexibility — bring any LLM provider, local or remote.

## Models

Works with any LLM: OpenAI, Anthropic, Google, Ollama, LM Studio, Together AI, Groq, and more. No vendor lock-in.

## Key Strengths

- **Fully open source** — Apache 2.0 license, active community
- **Any model** — connect any LLM provider, including local models
- **IDE native** — deep VS Code and JetBrains integration
- **Custom context providers** — pull context from docs, databases, APIs
- **Tab autocomplete** — inline completions with any model
- **Privacy** — can run 100% locally with Ollama

## Pricing

Free and open source. You pay only for LLM API usage (or nothing if using local models).

## Best For

- Privacy-conscious teams wanting local-only AI
- Developers who want full control over their AI stack
- Organizations that can't send code to external APIs
