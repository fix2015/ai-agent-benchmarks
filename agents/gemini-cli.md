# Gemini CLI

**Vendor:** Google
**Interface:** Terminal CLI
**Open Source:** Yes
**Repository:** https://github.com/google-gemini/gemini-cli

## Overview

Gemini CLI brings Google's Gemini models to the terminal with the largest context window available (1M tokens). It supports MCP, extensions, and has a generous free tier.

## Models

| Model | Context | Best For |
|-------|---------|----------|
| Gemini 2.5 Pro | 1M | Large codebases, comprehensive analysis |

## Key Strengths

- **1M token context** — largest context window of any coding agent
- **MCP support** — extensible via Model Context Protocol
- **Free tier** — generous free usage included
- **Extensions** — plugin system for custom functionality
- **Image input** — supports screenshots and diagrams
- **Web search** — integrated Google Search

## Pricing

Free tier available. API pricing for higher usage.

## Setup

```bash
npm install -g @anthropic-ai/gemini-cli
gemini
```

## Best For

- Very large codebases
- Budget-conscious developers (free tier)
- Google Cloud ecosystem users
