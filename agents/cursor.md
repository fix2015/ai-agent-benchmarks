# Cursor

**Vendor:** Anysphere
**Interface:** IDE (VS Code fork)
**Open Source:** No
**Website:** https://cursor.com

## Overview

Cursor is an AI-first IDE built as a fork of VS Code. It provides inline completions, chat-based generation, and an agent mode for autonomous multi-file edits.

## Models

Multi-model support including GPT-4o, Claude Sonnet/Opus, and proprietary models. Model availability depends on subscription tier.

## Key Strengths

- **Inline completions** — tab-to-accept code suggestions as you type
- **Agent mode** — autonomous multi-file task execution
- **Cursor Rules** — project-specific AI instructions (`.cursor/rules/`)
- **Familiar IDE** — VS Code compatibility, extensions work
- **Multi-model** — switch between providers easily
