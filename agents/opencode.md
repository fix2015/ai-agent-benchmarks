# OpenCode

**Vendor:** Anomaly
**Interface:** Terminal CLI
**Open Source:** Yes
**Repository:** https://github.com/anomalyco/opencode

## Overview

OpenCode is an open-source coding agent that rapidly gained 186k+ stars. It provides a terminal-based agentic coding experience with support for multiple LLM providers and a focus on developer experience.

## Models

Works with multiple providers: Anthropic (Claude), OpenAI (GPT-4), Google (Gemini), and local models.

## Key Strengths

- **Multi-provider** — connect any major LLM provider
- **TUI interface** — rich terminal UI with panels and navigation
- **MCP support** — extensible via Model Context Protocol
- **LSP integration** — language server protocol for better code intelligence
- **Session management** — persistent sessions with history
- **Custom tools** — define and register custom tools

## Pricing

Free and open source. Pay only for LLM API usage.

## Setup

```bash
curl -fsSL https://opencode.ai/install | bash
opencode
```

## Best For

- Developers wanting an alternative open-source CLI agent
- Teams wanting multi-provider flexibility
- Users who prefer a rich TUI over plain terminal output
